Deployment Instructions

This application uses Flask to connect your Python solver algorithms with this web interface. Follow these steps to deploy:

  1. Make sure all your Python files (app.py, simulated_annealing.py, sudoku_solver.py) are in the same directory
  2. Install Flask if you haven't already: pip install flask
  3. Run the app locally: python app.py
  4. Deploy to a platform like Heroku, Render, or PythonAnywhere by following their deployment instructions

For GitHub Pages users: GitHub Pages doesn't support Python execution. You'll need a different hosting platform that supports Python applications.
